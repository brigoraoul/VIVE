Introduction

Thank you for using Axies, {{ current_user.username }}!

Axies is a tool for the identification of personal values that are particularly important in a given context. Given a data set, the goal is to create a list of context-specific personal values by annotating some examples from the data. Axies consists of two phases: Exploration and Consolidation. Please start with the exploration and find the instructions for the individual phases on the respective pages.
